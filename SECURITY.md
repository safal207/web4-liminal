# Security Policy

## Supported Scope
This repository currently covers architectural RFC/spec artifacts and validation automation.

In-scope security issues include:
- CI workflow security misconfiguration
- accidental secrets in repository content
- integrity issues in validation scripts

## Reporting
- Do not open public issues for suspected vulnerabilities.
- Report privately with affected files, reproduction steps, and impact.

## Response Targets
- Acknowledgment: within 72 hours
- Triage: within 7 days
- Remediation plan: within 14 days for confirmed issues
